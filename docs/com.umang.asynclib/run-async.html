runAsync

fun <T> T.runAsync(exceptionHandler: ((Throwable) -> Unit)? = errorTrace, task: AsyncContext<T>.() -> Unit): Future<Unit>

Execute task asynchronously.

Parameters

exceptionHandler - optional exception handler. If defined, any exceptions thrown inside task will be passed to it. If not, exceptions will be ignored.

task - code to execute asynchronously.
